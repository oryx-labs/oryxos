# What is OryxOS

OryxOS is an enterprise Agent OS built on Java 21 and Spring Boot 3. It runs on your own Kubernetes cluster or servers as a unified platform for multiple business AI agents, providing shared channel access, LLM routing, tool execution, memory systems, and sandbox capabilities.

Unlike cloud-hosted agent solutions, OryxOS gives you full control over your infrastructure and data — no vendor lock-in, no data leaving your environment.
